## Overview

Build a modern, "Liquid Glass" styled Web GUI for Javinizer using FastAPI and TailwindCSS. The GUI will support all CLI features (sort, update, find) plus real-time logging and visual file management.

## Tech Stack

- **Backend:** FastAPI (Python), Uvicorn
- **Frontend:** Jinja2 Templates, HTMX (for interactions), TailwindCSS (styling), Alpine.js (local state)
- **Styling:** Custom "Liquid Glass" CSS (Glassmorphism)
- **Real-time:** Server-Sent Events (SSE) for logs
